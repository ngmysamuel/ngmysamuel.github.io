AutoCue

Completed over a period of 2.5 months, this project is done in native Swift and integrates with Google's vioce recognition services and Dropbox for cloud retrieval services.AutoCue recognises the current lines being rehearsed and automatically displays a cue card with the corresponding line of script. This allows the rehearsal to proceed smoothly with minimal hiccups.

In a later iteration of Autocue, a file explorer was implemented for users to more intuitively select their scripts on Dropbox.
